HNify An unofficial REST API for Hacker News Built using HackerNewsAPI

    Usage:

    Base URL: http://hnify.herokuapp.com

    Output: JSON

    • /get/top

      Returns stories from the front page of HN.

    • /get/newest

      Returns stories from the newest page of HN.

    • /get/best

      Returns stories from the best page of HN.

    Example:
